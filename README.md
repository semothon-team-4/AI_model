# 🧾 receipt_scan

Claude Vision AI를 활용해 세탁소 영수증 이미지를 분석하고,  
업체명 · 이용내역 · 금액 · 태그 분류 등을 자동으로 추출하는 모듈입니다.

---

## 📁 파일 구조

```
receipt_scan/
├── main.py              # FastAPI 서버 (API 모드)
├── cli.py               # CLI 실행 (카메라 촬영 / 파일 선택)
├── receipt_service.py   # Claude Vision 분석 핵심 로직
├── models.py            # Pydantic 데이터 모델 + 태그 그룹 빌더
└── requirements.txt
```

---

## ⚙️ 설치

```bash
pip install -r requirements.txt
```

### requirements.txt 주요 패키지

| 패키지 | 용도 |
|--------|------|
| `anthropic` | Claude Vision API 호출 |
| `fastapi` + `uvicorn` | REST API 서버 |
| `opencv-python` | 카메라 촬영 (CLI) |
| `pillow-heif` | 아이폰 HEIC 이미지 변환 |

## 🏷️ 태그 분류 기준

영수증 항목은 AI가 자동으로 아래 태그 중 하나로 분류합니다.

| 태그 | 이모지 | 분류 기준 |
|------|--------|-----------|
| 일반세탁 | 🧺 | 일반 의류 세탁 |
| 드라이클리닝 | 👔 | 드라이클리닝 의류 |
| 얼룩제거 | ✨ | 얼룩·오염 처리 |
| 신발관리 | 👟 | 신발 세탁·광택 |
| 명품케어 | 👜 | 항목명에 명품 브랜드 또는 "명품" 명시된 경우만 |
| 수선 | 🪡 | 수선·단추·지퍼 |
| 기타 | 📦 | 위 항목 외 |

> **명품케어 분류 원칙**: 영수증에 브랜드명(샤넬, 구찌, 루이비통 등)이나 "명품"이라는 단어가 명시적으로 적혀있을 때만 명품케어로 분류합니다. 표기가 없으면 다른 태그로 분류합니다.

# 👕 Cloth Grade — 옷 품질 등급 AI

Claude API를 활용해 옷 사진 한 장으로 소재, 오염도, 손상도를 분석하고 S/A/B/C 등급을 산정하는 개인 옷장 관리용 AI 서비스입니다.

---

## 📁 파일 구조

```
cloth_grade/
├── server.py        # FastAPI 서버 + Claude API 호출
├── index.html       # 브라우저 UI (카메라/갤러리 업로드)
├── requirements.txt # 의존성 패키지
└── .env             # API KEY 설정 (git 제외)
```

---

## ⚙️ 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. API KEY 설정
`.env` 파일에 Anthropic API KEY 입력
```
ANTHROPIC_API_KEY=sk-ant-여기에입력
```
> API KEY 발급: https://console.anthropic.com

### 3. 서버 실행
```bash
uvicorn server:app --reload
```

### 4. 브라우저 접속
```
http://localhost:8000
```

## 🔍 분석 항목

| 항목 | 설명 |
|------|------|
| 옷 종류 | 청바지, 티셔츠, 니트, 코트 등 |
| 소재 | 데님, 면, 폴리에스테르, 울 등 |
| 빈티지 여부 | 의도적 워싱/에이징 디자인 감지 |
| 오염도 | 실제 오염 영역 비율 (0~100%) |
| 손상도 | 구멍/뜯김/마모 영역 비율 (0~100%) |
| 종합 점수 | 오염도 × 0.4 + 손상도 × 0.6 |
| 등급 | S / A / B / C |
| 관리 가이드 | 세탁 권장/필요, 수선 권장/필요, 처분 권장 |
| 보관 팁 | 소재별 맞춤 보관 방법 |

## 🏅 등급 기준

### 손상도 우선 규칙
> 손상도 **36% 이상**이면 오염도에 관계없이 **무조건 C등급**
> (손상은 세탁으로 해결 불가)

### 종합 점수 기반 등급

```
종합점수 = 오염도 × 0.4 + 손상도 × 0.6
```

| 등급 | 종합점수 | 의미 | 추천 행동 |
|------|---------|------|---------|
| S | 0 ~ 10 | 새것에 가까움 | 그대로 착용 |
| A | 11 ~ 25 | 약간의 사용감 | 그대로 착용 (세탁/수선 권장) |
| B | 26 ~ 45 | 눈에 띄는 결함 | 세탁/수선 후 착용 |
| C | 46+ | 심각한 결함 | 처분 권장 |

### 관리 태그

| 태그 | 조건 |
|------|------|
| ✅ 관리 불필요 | S등급 또는 오염/손상 모두 낮음 |
| 🧺 세탁 권장 | A등급 + 오염도 6% 이상 |
| 🧺 세탁 필요 | B등급 + 오염도 6% 이상 |
| 🪡 수선 권장 | A등급 + 손상도 6% 이상 |
| 🪡 수선 필요 | B등급 + 손상도 6% 이상 |
| 🔴 처분 권장 | C등급 |

## 🌟 주요 기능

### 빈티지 감지
의도적으로 낡게 만든 워싱, 에이징, 페이딩, 디스트레스드 디자인을 감지해 결함이 아닌 디자인으로 인정합니다. 실제 오염이나 의도치 않은 손상만 결함으로 판단합니다.

### 사진 품질 체크
분석이 불가능한 수준(극도로 어둡거나 심하게 흔들린 사진)일 경우 재촬영을 안내합니다. 구겨져 있거나 배경이 복잡한 사진도 최대한 분석합니다.

### 촬영 가이드
- ☀️ 밝은 곳에서 찍기
- 📐 최대한 펼쳐서 정면으로
- 🟦 단색 배경이면 더 정확

## 🔌 API 명세 (팀원 백엔드 연동용)

### 옷 품질 분석
```
POST /analyze
Content-Type: multipart/form-data
Body: file (image/jpeg, image/png, image/webp, 최대 10MB)
```

#### 응답 예시
```json
{
  "photo_quality": "good",
  "photo_warn": null,
  "clothing": "청바지",
  "fabric": "데님",
  "is_vintage": false,
  "vintage_reason": null,
  "stain": 15,
  "damage": 10,
  "total_score": 12.0,
  "grade": "A",
  "need_wash": true,
  "need_repair": false,
  "action": "세탁 권장",
  "storage_tip": "청바지는 뒤집어서 찬물에 세탁하고 직사광선을 피해 보관하세요.",
  "reason": "약간의 오염이 있으나 손상은 없어 세탁 후 착용 가능한 상태"
}
```

### 서버 상태 확인
```
GET /health
```
